Для полной работоспособности программы, нужно перенести файлы из папки "Files" в папку сборки программы в папку "app".

Если вы ни разу не собирали программу, то вы не сможете найти папку сборки.

Примерное название папки сборки: "build-Project_Work-Desktop_Qt_5_12_4_MinGW_64_bit-Debug"
Следовательно, в этой папке существует папка "app" куда нужно закидывать текстовые файлы из папки "Files" проекта.

Спасибо за понимание!
